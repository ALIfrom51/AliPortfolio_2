## 🚀 Getting Started (Copy-Paste Ready)

### Step 1: Create React Project
```bash
npx create-react-app ali-portfolio
cd ali-portfolio
```

### Step 2: Install Dependencies
```bash
npm install framer-motion lucide-react
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Step 3: Copy Files
- Copy content of `AliRebbouh-Portfolio.jsx` → `src/App.jsx`
- Copy content of `tailwind.config.js` → `tailwind.config.js`
- Copy content of `styles.css` → `src/index.css`
- Copy content of `vite.config.js` → `vite.config.js`

### Step 4: Run Locally
```bash
npm run dev
```

### Step 5: Visit
```
http://localhost:5173
```

## 🆘 Troubleshooting

### Issue: "Module not found"
```bash
npm install
npm run dev
```

### Issue: "Styles not loading"
```bash
# Restart dev server
npm run dev

# Or rebuild Tailwind
npm run build
```

### Issue: "Images not showing"
- Check image paths in `public/` folder
- Use relative paths: `./image.jpg`

### Issue: "Form not working"
- Follow form integration guide in CUSTOMIZATION_GUIDE.md
- Test locally before deploying

### Issue: "Animations laggy"
- Check GPU acceleration in DevTools
- Test on different browser
